= Aggiorna le credenziali di vCenter Server
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
È possibile aggiornare le credenziali dell'istanza di vCenter Server tramite la console di manutenzione.

.Prima di iniziare
È necessario disporre delle credenziali di accesso per gli utenti di manutenzione.

.A proposito di questa attività
Se hai modificato le credenziali di vCenter Server dopo la distribuzione, aggiornale utilizzando questa procedura.

.Fasi
. Dal server vCenter, aprire una console per il provider VASA.
. Accedere come utente di manutenzione.
. Inserire `2` per selezionare il menu Configurazione di sistema.
. Entra `8` per modificare le credenziali di vCenter.
