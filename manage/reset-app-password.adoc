= Reimpostare la password utente dell'applicazione
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Seguire questi passaggi per reimpostare la password utente dell'applicazione necessaria per la registrazione del provider SRA e VASA con vCenter Server utilizzando gli ONTAP tools for VMware vSphere.

.Fasi
. Apri un browser web e vai su: `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. Accedere utilizzando le credenziali di amministratore configurate durante la distribuzione degli strumenti ONTAP .
. Dalla barra laterale, seleziona *Impostazioni*.
. Nella pagina *Credenziali VASA/SRA*, seleziona *Reimposta password*.
. Inserisci e conferma la nuova password.
. Selezionare *Reimposta* per applicare la nuova password.
